# PDF → JSON Extractor

✅ Live site: https://wajahathssn.github.io/

This repo hosts a simple web UI (GitHub Pages) for uploading a PDF and running extraction to JSON (via your backend API).

## How to use
1. Open the live site link above.
2. Upload a PDF.
3. Click **Upload & Index**.
4. Enter your query / schema.
5. Click **Run Extraction**.
6. Copy or download the JSON output.

## Where the code is
- Website UI: `index.html`
- Update the backend URL by editing `API_BASE` inside `index.html`.
